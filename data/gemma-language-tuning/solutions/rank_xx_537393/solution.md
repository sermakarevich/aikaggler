Listing a few last year's Google – AI Assistants for Data Tasks with Gemma winning solutions/honorable mentions that might be useful to look into to understand how the notebooks were organized and explained. 

1. [Build AI Agents With Google's LLM Gemma](https://www.kaggle.com/code/sitaberete/build-ai-agents-with-google-s-llm-gemma),  the author uses techniques such as QLoRa, RAG, and ReAct to build an AI Assistant that specializes in teaching or explaining data science concepts.
2. https://www.kaggle.com/code/woosungyoon/teacher-gemma-heuristic-algorithm
3. https://www.kaggle.com/code/hserdaraltan/data-science-with-gemma-llamaindex-and-pinecone
4. [PyGEM: A Chatbot for Python Questions using Gemma](https://www.kaggle.com/code/davidtroxellucla/pygem-a-chatbot-for-python-questions-using-gemma), the author combines techniques such as LoRa and RAG with a variety of thoughtfully chosen datasets in order to create a system called PyGem that can be used to answer common questions about the Python programming language.
5. [KaggleBot - Gemma-7b-it RAG w few-shot prompting](https://www.kaggle.com/code/vbookshelf/kagglebot-gemma-7b-it-rag-w-few-shot-prompting), the author uses a variety of carefully curated datasets along with few-shot prompting to setup a robust and reliable RAG system that can be used to answer common questions about the Kaggle platform. Both few-shot prompting and general heuristics were used in order to improve the structure and style of the outputs, and a representative sample of 25 prompt/output pairs were provided for evaluation. We were really impressed at how accurately it responded to our questions and how well it knew when to decline to respond.
