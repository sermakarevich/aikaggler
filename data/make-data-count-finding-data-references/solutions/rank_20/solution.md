**Accession IDs**

* Extracted candidates using regex patterns.
* Fine-tuned **DeBERTa** to filter out false positives.
* Certain ID types always mapped directly to **Secondary**.
* For the rest, fine-tuned **Qwen2.5-3B (base)** with QLoRA:

  * Input = identifier + context in a structured template with cues.
  * Representation = global mean + target-span mean pooling.
  * Added a **span-gated GELU classifier head**.

**DOIs**

* Extracted DOI prefixes with regex (`10.\d{4,9}`) and removed non-data prefixes.
* Used **Qwen3-4B** to detect DOIs in noisy windows.
* Applied **DeBERTa** to filter false positives.
* For classification:

  * Extracted paper title and author list with **Qwen2.5-14B**.
  * Combined with context as input to **Light-R1-14B-awq**.
  * Decision rules:

    1. Check if DOI is data vs non-data.
    2. If authors available, decide based on individual overlap.
    3. If authors missing, use context cues.
    4. If still unclear, assign None.

**Data & Labels**

* Training data from MDC corpus v3.0 and v4.1.
* Labels generated by consensus of **GPT-5, Qwen3-30B-thinking, Qwen2.5-72B, LLaMA-70B**.
* Planned Light-R1 fine-tuning for DOIs, but could not complete due to label issues.
